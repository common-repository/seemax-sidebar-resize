== Description ==
A simple button interface to make the WordPress Gutenberg sidebar resizable.

### Key Features
- Works out of the box
- Simple interface


== Installation ==
### Requirements
- PHP 5.6 or higher is recommended.

### Steps to Install:
1. Navigate to the WordPress dashboard.
2. Go to 'Plugins' -> 'Add New'.
3. Search for 'SeeMax Resizable Editor Sidebar'.
4. Click 'Install Now' and then 'Activate' to activate the plugin.
5. You can now resize the Gutenberg sidebar to suit your preferences.
